Types of Css Selectors

A CSS selector is the first part of a CSS Rule. It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them.

The element or elements which are selected by the selector are referred to as the subject of the selector.

  • Type Selector

    Element selectors select elements by tag name, i.e. type, such as div or ul. The style applies to all elements of that type.

    Examples:div,div ul,p+h4

  • Class Selector

  • Id Selector

    The Id selector selects HTML elements with a specific id attribute.
